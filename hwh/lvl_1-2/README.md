# Hashtable (frequency dictionary)
============================================================================
## To run this project
1. clone this repo using command
```
  $ git clone git@github.com:klauchek/C-contests.git
```
2. cd to the `hwh/lvl_1-2` and build project using command
```
  $ make
```
3. run program
```
  $ ./hwh_lvl_1-2
```
